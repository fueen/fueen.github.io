面试题：$和#的区别？

$在mybais中代表拼接的意思,将传入的数据之间显示成为sql语句

比如：select *from user where id =${id}传给id的值为username,username的值为11

相当于select * user where id = username

#是取后面的值

比如：select *from user where id =${id}传给id的值为username,username的值为11

相当于select * user where id = 11

大多数情况都是用#
