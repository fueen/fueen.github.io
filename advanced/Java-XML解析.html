发表于 2019-08-08 | 分类于 Java进阶 | | 阅读量 次

XML

概念

可扩展的标记语言 和HTML类似 版本1.0

作为通用的数据交换格式

1.通常用来作为配置文件(框架的配置文件都会使用XML)

2.与平台无关，可作为数据交换的一个过渡

3.文档清晰，方便管理数据(properties 键值对存在，XML 文档存在，相当于一个树形结构)

语法

单标签 只有一个

双标签 有开始有结尾

标签名字

&lt <符号 &gt >符号

对于要转义的地方

注意事项

1.一个XML只能有一个根节点

2.正确嵌套，层级分明

3.严格区分大小写

4.标签名自定义

5.特殊字符需要进行转义

XML约束

1.主要约束XML中能够使用哪些标签

2.哪些标签是必须的

3.标签的顺序是怎样的

4.可以给一些智能提示

两种约束：dtd(功能稍微简单) Schema(功能强大，本身也是一个XML)
